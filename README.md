## 功能特点

### 🎯 一键启动
- 快速启动/停止 OpenClaw Gateway 服务
- 实时监控服务运行状态
- 自动检测并显示当前版本信息

### 🔧 丰富的管理功能
- **自动修复**：一键诊断并修复常见配置问题
- **更新检查**：支持一键升级到最新版本
- **终端面板**：内置 PowerShell 终端，方便执行命令
- **运行日志**：实时查看服务日志，了解运行状态
